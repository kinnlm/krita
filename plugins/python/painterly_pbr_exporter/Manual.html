Painterly PBR Exporter

The Painterly PBR exporter looks for Material Groups in the current document and writes PBR-ready texture sets for each channel stack (BaseColor, Height, Normal, Roughness and Metallic). Optional Occlusion layers are supported for ORM packing.

Using the exporter

  1. Open the document that contains one or more Material Groups.
  2. Choose File → Export → Export PBR Set….
  3. Select an output folder and optional filename prefix.
  4. Choose the desired bit depth for Height and Roughness/Metallic maps.
  5. Enable the options that match your target renderer (Flip Green for DirectX, Bake Normals, Pack ORM).
  6. Press OK to write the textures and material.yaml manifest.

Outputs

  • BaseColor: 8-bit sRGB PNG.
  • Height: 16-bit PNG or 16-bit EXR depending on your choice.
  • Normal: 8-bit tangent-space PNG. Can be baked from Height on export.
  • Roughness and Metallic: 8-bit or 16-bit linear PNG.
  • ORM: Optional RGB pack (R=Occlusion, G=Roughness, B=Metallic).

The exporter preserves the .kra document untouched; all processing happens on layer duplicates in temporary documents.
